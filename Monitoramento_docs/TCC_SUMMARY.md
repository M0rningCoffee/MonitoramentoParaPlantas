Título: Monitoramento de Plantas com sensores de umidade — Aplicativo móvel e backend local

Resumo:
Sistema que permite pequenos produtores/usuários monitorar a umidade do solo, cadastrar plantas e visualizar histórico. Aplicativo em React Native (Expo) e backend simples em PHP com MySQL.

Componentes:
- App móvel com telas: login, cadastro, esqueci senha, dashboard.
- Backend PHP que registra leituras e autentica usuários.
- Banco de dados MySQL com tabelas: users, solo, planta, log.

Contribuições:
- Protótipo funcional para TCC com integração app ↔ backend local.
- Recomendação para produção: migrar backend para API RESTful com autenticação JWT e deploy em VPS/Heroku/AWS.
